# Template Files

Copy these into your Go project root to set up Claude Code + tooling.

## Files in this directory

- `CLAUDE.md` — project instructions for Claude Code
- `CLAUDE.md.microservice` — variant for microservices
- `CLAUDE.md.cli` — variant for CLI tools
- `.golangci.yml` — strict linter config (matches skills)
- `Makefile` — canonical build targets
- `docker-compose.dev.yaml` — local development stack
- `.github/workflows/ci.yaml` — GitHub Actions CI pipeline
